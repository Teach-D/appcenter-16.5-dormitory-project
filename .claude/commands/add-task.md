현재 이슈에 새 작업을 추가해줘: $ARGUMENTS

## 진행 순서

### 1단계: 현재 이슈 확인
`.claude/issues.md`를 읽어서 현재 작업 이슈 번호 파악.
GitHub MCP `get_issue`로 최신 이슈 본문 조회.

### 2단계: 작업 추가 확인
추가할 작업 내용을 출력하고 확인:
```
다음 작업을 이슈 #{번호}에 추가할까요?

- [ ] {추가할 작업}
```

### 3단계: 이슈 업데이트 (GitHub MCP)
`update_issue` 도구로 기존 본문의 작업 목록에 새 항목 추가:
- 기존 `## 작업 목록` 섹션 끝에 `- [ ] {새 작업}` 추가

### 4단계: 로컬 issues.md 업데이트
`.claude/issues.md`의 작업 목록에도 동일하게 추가.

### 5단계: 완료 안내
```
작업이 추가되었습니다.

#{번호} 전체 작업 목록:
- [x] {완료된 작업}
- [ ] {기존 작업}
- [ ] {방금 추가한 작업} ← 새로 추가됨

`/work`로 계속 진행하세요.
```
